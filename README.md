# Yle Share Buttons 
Faster and more secure share buttons

No dependencies.

Uses somedata API for counting the total number of shares.

## Status
Do not use in production yet.

## Building

    grunt

## Running

    cd dist
    node server.js

## Supported Social Platforms
- Facebook
- Twitter
- LinkedIn
- Google+
- Pinterest
